# Release Checklist

Every API Vault release must explicitly confirm these steps before distribution. 
There are **no shortcuts**.

* [ ] CI Green
* [ ] Analyze Clean
* [ ] Unit Tests 100%
* [ ] Integration Tests 100%
* [ ] Red Team Tests 100%
* [ ] Performance Budgets Met
* [ ] Dependency Audit Complete
* [ ] No Critical CVEs
* [ ] Backup Compatibility Verified
* [ ] Manual Unlock Tested
* [ ] Biometric Unlock Tested
* [ ] Fresh Install Tested
* [ ] Upgrade Tested
* [ ] Export/Import Tested
* [ ] Architecture Invariants Hold
