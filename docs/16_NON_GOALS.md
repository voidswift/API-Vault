# NON-GOALS

To protect API Vault from scope creep, these features are explicitly declared as out of scope. If a feature proposal aligns with anything on this list, it should be rejected.

API Vault is **NOT**:
* ❌ A cloud password manager (like 1Password or LastPass).
* ❌ A browser password autofill tool.
* ❌ A source-code editor.
* ❌ A Git client.
* ❌ A CI/CD platform.
* ❌ A replacement for Bitwarden's general-purpose usage.
* ❌ An API gateway.
* ❌ A network traffic sniffer.
* ❌ A social network for developers.

The app must remain focused purely on being a local, secure Developer Security Hub for managing API keys, cryptographic tools, and developer secrets.
